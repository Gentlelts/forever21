# forever21

## 运行
```
ng serve --open
```
#### pages页面生成
ng g page(pg) pages/xxx
页面生成到pages下,会自动添加到app.modules里,只需在app-routing里添加对应的路由即可

### component生成
ng g c(components) xxx
页面生成到app下,会自动添加到app.modules里,只需在app-routing里添加对应的路由即可
